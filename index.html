MY WORKFLOW

Each project is different, but the process is pretty much the same each time. Checkout the Process

01.

UI-UX FIGMA

Find Figma design based on my research, ensure it's free for community use, establish typography, color palette, and breakpoints. Move on to Photoshop.

02.

PHOTOSHOP

After finalizing typography and Figma template, work on SVG and icons, scale images, make corrections, and prepare for coding.

03.

THE BIG3 (HTML CSS & JS)

Translate designs into code with HTML emphasizing semantic structure, CSS focusing on specificity and typography, and JavaScript for functionality, DOM manipulation, and local storage handling.

04.

RESPONSIVE & MINIFY

Focus on performance optimization by reducing file sizes, using lazy loading for images, CSS sprites if needed, and ensuring selectors are appropriate. Consider using React or other frameworks based on project requirements, excluding Angular and Java.

05.

GOOGLE INSIGHTS

Conduct Google Insights checks for optimization, performance improvements, and SEO suggestions.
